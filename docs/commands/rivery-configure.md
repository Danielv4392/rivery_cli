
# rivery configure

Configure new profile and the authentication.

## Usage

```
Usage: rivery configure [OPTIONS]
```

## Options
* `profile`: 
  * Type: STRING 
  * Default: `none`
  * Usage: `--profile`

  


* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
