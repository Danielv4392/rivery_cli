
# profiles get

Get the profiles list from the auth config file.

## Usage

```
Usage: profiles get [OPTIONS]
```

## Results

```shell
+----------------+-----------------------------+-------------+---------------+
|  Profile Name  |             Host            |    Region   |     Token     |
+----------------+-----------------------------+-------------+---------------+
|    default     |  https://console.rivery.io  |  us-east-2  |  ******AAAaa  |
+----------------+-----------------------------+-------------+---------------+
```

## Options
* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
